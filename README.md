## 🛠 기술 스택

| 분류        | 기술                                                  |
|-------------|------------------------------------------------------|
| **Frontend** | React 18<br>React Router DOM<br>JavaScript<br>CSS Modules |
| **디자인**     | CSS3<br>미디어 쿼리를 활용한 반응형 웹 구현              |
| **배포**       | GitHub Pages (`gh-pages` 브랜치로 정적 파일 배포)         |
| **버전 관리**  | Git / GitHub                                        |
